# Windows Vista Portfolio

A personal portfolio web app inspired by Windows Vista, built with HTML, CSS, and JavaScript. It features draggable windows, a start menu, and interactive desktop icons to showcase projects, skills, and contact information.

## Features
- Desktop-like UI with multiple windows (About, Skills, Projects, Documents, Control Panel, Games, etc.)
- Dynamic project cards and details
- Minesweeper mini-game
- Downloadable resume (CV)
- Theme toggle (dark/light)
